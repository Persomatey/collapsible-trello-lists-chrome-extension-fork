# Better Collapsible Trello Lists Chrome Extension
This is an improvement of the original [Collapsible Trello Lists](https://chromewebstore.google.com/detail/collapsible-trello-lists/kejemfjfgoaekplmemlkjljcfloiemho) Chrome Extension by [Jeremy Laskar](https://webrender.net/) which is no longer available in the Chrome Webstore but here still be accessed on [Chrome-Stats](https://chrome-stats.com/d/kejemfjfgoaekplmemlkjljcfloiemho/download). 

## Features 
- Adds a button above each Trello list that allows you to collapse the column
	- This is the original functionality
- Displays the count of cards currently in the collapsed list

## Installation 
1. Download the latest release
	- For the latest:
		- `https://github.com/Persomatey/better-collapsible-trello-lists-chrome-extension/releases/latest` 
2. Download the .zip file in Assets 
	- Should titled something like: 
		- `better-collapsible-trello-lists-chrome-extension-v0.0.14.zip`
3. Extract the .zip to a separate folder
4. In your Chrome browser, navigate to Settings > Extensions > Manage Extensions
5. Click "Load Unpacked" and select your extracted folder
